# History of Mathematics

## 🌱 1. Survival Maths (Beginning of thinking)

**Why it started:**
Humans needed to:

* Count animals
* Share food
* Track time

👉 Maths = **tool for survival**

Early ideas:

* Counting with fingers
* Basic numbers
* Simple patterns (day/night, seasons)

## 🏺 2. Civilizations Build Maths

### Mesopotamia

* First number systems
* Base 60 system (why 60 seconds, 60 minutes!)

### Ancient Egypt

* Geometry for land measurement
* Building pyramids

👉 Maths becomes: **practical problem-solving**

## 📐 3. Logical Maths Begins (Big shift)

### Pythagoras

* Numbers have patterns
* Famous theorem (triangles)

### Euclid

* Wrote *Elements*
* Introduced proofs

👉 Maths becomes: **logic + reasoning**

## 🧮 4. India & Zero (Game changer 🚀)

### Ancient India

### Aryabhata

* Work on algebra, astronomy

### Brahmagupta

* Introduced **zero as a number**
* Rules for negative numbers

👉 This is HUGE:
Without zero → no computers today

## 🌍 5. Algebra & Expansion

### Al-Khwarizmi

* Father of algebra
* Step-by-step problem solving

👉 Word “algorithm” comes from his name

## 🔬 6. Calculus (Understanding change)

### Isaac Newton

### Gottfried Wilhelm Leibniz

* Motion, speed, change
* Used in physics, engineering

👉 Maths becomes: **tool to understand the universe**

## 💻 7. Modern Maths → Computer Science

* Logic → programming
* Algebra → data structures
* Probability → AI & ML

👉 Maths becomes: **thinking engine of technology**

# NUMBER SYSTEM

## 1. Types of Numbers

<img width="541" height="555" alt="image" src="https://github.com/user-attachments/assets/b40cad0e-bf53-43f6-9727-ddbdf22135bf" />

### **Natural Numbers (N)**
  → 1, 2, 3, 4, …
  → Counting numbers

### **Whole Numbers (W)**
  → 0, 1, 2, 3, …
  → Natural + 0

### **Integers (Z)**
  → … -3, -2, -1, 0, 1, 2, 3 …
  → Positive + Negative + 0

### **Rational Numbers (Q)**
  → Can be written as p/q , **natural, whole, integers, fractions, terminating and repeating decimals in p/q form (q ≠ 0).**
  → Example: 1/2, -3/4, 2

### **Irrational Numbers**
  → Cannot be written as p/q, **non-terminating, non-repeating decimals**
  → Example: √2, π

### **Real Numbers (R)**
  → Rational + Irrational

## 2. Number Line Thinking 

* Every number has a position
* Left → Negative
* Right → Positive
* Distance from 0 = value

Helps in: Comparisons, Absolute value, Logic building

---

## 3. Basic Properties 

* **Closure** → a + b is same type
* **Commutative** → a + b = b + a
* **Associative** → (a + b) + c = a + (b + c)
* **Distributive** → a(b + c) = ab + ac

---

## 4. Even & Odd

* Even → divisible by 2
* Odd → not divisible by 2

Patterns:

* Even + Even = Even
* Odd + Odd = Even
* Even + Odd = Odd

---

## 5. Prime, Composite & Co-prime

* **Prime** → only 2 factors (1 & itself)
  ✔ Example: 2, 3, 5, 7

* **Composite** → more than 2 factors
  ✔ Example: 4, 6, 8

* **Co-prime** → two numbers with HCF = 1
  ✔ Example: (8, 15), (9, 10)

👉 1 is neither prime nor composite


---

## 6. Factors & Multiples

* Factor → divides completely
* Multiple → result of multiplication

Example:

* 6 → factors: 1,2,3,6
* 6 → multiples: 6,12,18…

---

## 7. HCF & LCM

* **HCF (GCD)** → largest common factor
* **LCM** → smallest common multiple

👉 Key relation:

```
HCF × LCM = Product of numbers
```

---

## 8. Divisibility Rules (Quick Thinking)

* 2 → last digit even
* 3 → sum of digits divisible by 3
* 5 → ends in 0 or 5
* 9 → sum of digits divisible by 9
* 10 → ends in 0

---

## 9. Representation of Numbers

* Decimal (base 10)
* Binary (base 2) → important for coding
* Octal (base 8)
* Hexadecimal (base 16)

---

## 10. Special Concepts

* Absolute Value → |x| = distance from 0
* Co-prime → HCF = 1
* Zero → neutral element
* Negative numbers → opposite direction
